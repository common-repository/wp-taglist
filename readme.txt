== Description ==

This Plugin create a taglist overview page with most important tags to each letter category. This plugin will also create a page for each letter, who display all tags of this letter category. 

At the Adminpanel, you can do some setting for this Plugin!

Sample:
-------------------
	Tags from A - Z
    * A Tags
          o Anfang(3)
          o Antrag(2)
          o Apples(2)
          o Aufbau(1)
          o Aufmerksamkeit(1) 
    * B Tags
          o Bildschirm(3)
          o Bestandsaufnahme(1)
          o Besucher(1)
          o Blicke(1)
          o Bitkom(1) 
    * C Tags
		...
		
---------------------------------------------------------------------------


Dieses Plugin erstellt eine &Uuml;bersichtsseite mit den wichtigsten Tags zu einem Buchstaben. Au&szlig;erdem erstellt das Plugin zu jedem Buchstaben eine Seite mit allen Tags zum Buchstaben.

Im Adminbereich k&ouml;nnen diverse Einstellungen f&uuml;r dieses Plugin vorgenommen werden.

Beispiel:
-------------------
	Alle Tags von A - Z
    * A Tags
          o Anfang(3)
          o Antrag(2)
          o Apples(2)
          o Aufbau(1)
          o Aufmerksamkeit(1) 
    * B Tags
          o Bildschirm(3)
          o Bestandsaufnahme(1)
          o Besucher(1)
          o Blicke(1)
          o Bitkom(1) 
    * C Tags
		...
		

== Installation ==

Download, Activate, make your Config under Settings -> Wp Taglist and have fun.
